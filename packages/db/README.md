# packages/db

Schema Drizzle, migraciones y utilidades de base de datos.

PostgreSQL sera la fuente de verdad para usuarios, sesiones, mesas, manos, eventos, participantes y ledger de fichas virtuales.

Comandos:

```bash
pnpm --filter @neon-poker/db db:generate
pnpm --filter @neon-poker/db db:migrate
```
